#Giftza

# Introduction

Giftza is a project for crawling the data from [giftza.co](https://www.giftza.co/). 

Giftza is written base on Scrapy.

# Requirements

* Python 3

# Install

Clone the project to your device, then:

    $ pip install -r requirements.txt
